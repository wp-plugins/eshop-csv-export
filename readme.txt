== Description ==

This is a free (export only) version of CSV Imp.

[CSV Imp](http://csv-imp.paulswebsolutions.com 'Learn more about CSV Imp') is a WordPress plugin for importing and exporting [eShop](http://wordpress.org/extend/plugins/eshop/ 'eShop Plugin Page') products via the CSV file format. If you need to create, modify, or delete dozens or even thousands of products then CSV Imp is intended to save you a lot of time and effort.

CSV Imp also supports embedded images and thumbnails, tags and categories, custom fields, and much more.
== Installation ==

1. Either use the automatic installer, or download the zip and extract to your 'wp-content/plugins' folder.
2. Open the Tools menu on the left side of your Wordpress dashboard
3. Click on 'eShop CSV Export'

== Frequently Asked Questions ==

= How can I import my data? =

This is an export only demonstration of [CSV Imp](http://csv-imp.paulswebsolutions.com 'Learn more about CSV Imp').  You'll need to download CSV Imp to be able to import data (you can create new products, modify existing ones, and also delete).
